>> HTML5 <<



HTML corresponde a las siglas en inglés de "HiperText Markup Languaje" que traducido al español es "Lenguaje de Marcas de Hipertexto" es considerado un lenguaje que se encuentra constituido por tres caracteristicas básicas: estructura, estilo y funcionalidad.

Todo HTML se basa en un concepto basado en etiquetas, que es una manera de comunicarse con el navegador, es decir, que el HTML a diferencia de los lenguajes de scripting o programación que usan secuencias de comandos para realizar procesos, utiliza diferentes etiquetas para identificar el contenido y así comunicarsecon el navegador.



Ahora bien el HTML5 es considerado el producto de la combinación de HTML, CSS y JavaScript. Estas tecnologías son altamente dependientes y actuan como una sola unidad organizada bajo la especificación de HTML5.
Html está a cargo de la estructura, CSS presenta esa estructura y su contenido en la pantalla, mientras JavaScript hace el resto.
