HTML5

HTML corresponde a las siglas en inglés de "HiperText Markup Languaje" que traducido al español es "Lenguaje de Marcas de Hipertexto" es considerado un lenguaje que se encuentra constituido por tres caracteristicas básicas: estructura, estilo y funcionalidad.

Todo HTML se basa en un concepto basado en etiquetas, que es una manera de comunicarse con el navegador, es decir, que el HTML a diferencia de los lenguajes de scripting o programación que usan secuencias de comandos para realizar procesos, utiliza diferentes etiquetas para identificar el contenido y así comunicarsecon el navegador.

Lenguaje sencillo, para la elaboración de sitios web, definido como una forma de modelar la información para las personas. Se basa en el manejo de etiquetas las cuales hacen posible comunicarse con el navegador, y este comprenda las respectivas órdenes que se desea ejecutar.

La información principal es el contenido de más alta prioridad, en el de una una página web, a su vez es una sección muy fexible, por lo cual es totalmente adaptable a las necesidades de la pagina. Pero ¿cómo hacemos que la información principal abarque todo lo que queramos introducir?. Esto se realiza insertando más columnas, dividiendo cada columna en bloques más pequeños o generando diferentes distribuciones y combinaciones. para ello utilizamos:

  • * ASIDE
  • * ARTICLE
  • * FOOTER
